Squirrel's Adventure

December 2025
Unity and C#
×

I made this video game as my final project for one of my college courses about game making and it was the first video game that I have ever made of this size. My intention for this project was to create a game that anyone of any age could play and enjoy, similar to other nostalgic video games that I grew up playing with my friends and family. Creating this game was my first experience making a game using Unity and C#, and throughout the process I had to learn a lot about both, in order to realize my vision. I have always loved video games and gaming in general, so creating this game has inspired me to continue to pursue game making.

Hypnotic Florals

December 2025
P5.js
×
×

I made this piece as my final project for one of my college courses about interactive media and it was my second time making an artwork using text-based code. The idea for this piece was inspired by kaleidoscopic imagery, as well as wanting to create something visually stimulating that also had optional interactive elements. I wanted the user to be mesmerized by the flowing shapes and colours, as they move around and blend into one another. While making this project, I learned more about coding using P5.js and interactive artworks. In the future, I'd like to continue to pursue creating interactive media using code, so this experience was very helpful in showing me the possibilities of what I might be able to do.

Colour Collisions

May 2025
Max
×

I made this piece as my final project for one of my college courses about interactive media and it was the first artwork I ever made using Max. I have always liked the idea of generative and randomized artworks, because I find it interesting to look at an artwork knowing that it will have a different result every time someone sees it. I wanted the user to be captivated by the shapes and colours constantly in motion and merging together as they connect. Throughout the process of making this piece, I learned a lot more about the capabilities of Max and got ideas of other things that I could use the software to create in the future.

Cephalonian Association Website

February 2025
Google Sites
×

I made this website for a non-profit cultural association at which I volunteer. We spoke about expanding our online presence and mentioned how it would be nice to have a website. I had never made a website before, but I had been interested in web design for a while, so I decided to give it a try. The intention for the website is to inform viewers about what the association is and does, as well as let people know about past and upcoming events. This experience inspired me to learn more about web design, as well as UI and UX. It also got me interested in learning HTML and CSS, in order to one day create other websites without needing to use an online website builder.

Sabrina Rath and Zeina Hussein

I made this board game as my final project for one of my college courses about game design. This game was created in collaboration with two of my classmates, one of which was very interested in Greek mythology, so we drew inspiration from the Odyssey for the plot of our game. We wanted to make this board game accessible and fun for all ages, with fun and simple artwork that adds to the experience. Throughout making this game, I learned what elements are required to make a good board game. In the future, I would like to continue to explore game design with both physical and digital games.

Digital Ruin

December 2024
Adobe Photoshop
×

I made this piece as my final project for one of my college courses about digital art, and it was one of the first projects I made using Photoshop. The assignment was to create an artwork that depicted the future, so I decided to create a representation of a dystopian future ruled by technology. I wanted the viewer to reflect on the role technology has in our society, how it continues to grow and the possible consequences. Throughout the process of creating this piece, I learned more about Photoshop techniques and how to convey messages through art. In the future, I would like to continue to use applications like Photoshop to create new artworks and further explore graphic design.
